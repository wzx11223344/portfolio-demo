# 实战作品集 · Interactive Portfolio Demo

交互式能力展示平台 — 单文件纯HTML/CSS/JS，展示技术能力和算法实现。

## 功能

- **作品集全貌**：15台MCP服务器 (~370工具)、12个效率技能 (17,138行)、58个SkillHub发布
- **算法可视化演示**：
  - 排序算法实时对比（冒泡/快速/归并/堆排序）
  - 路径规划算法（BFS/DFS/A*）
  - 递归分形树（毕达哥拉斯树/Koch雪花/谢尔宾斯基三角/龙曲线）
- **粒子背景动画** + 滚动揭示动画
- **GitHub实时活动流**
- **项目演进时间线**
- 深色主题 · 赛博霓虹风格

## 使用

直接打开 `index.html` 即可在浏览器中查看。

或部署到任意静态服务器：
```bash
python -m http.server 8888
```

## 技术栈

HTML5 · CSS3 · JavaScript (纯原生，零依赖，无框架)
